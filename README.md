# TodoList

처음 Kotlin을 배울 때 만들어 본 어플입니다.

가장 간단하고 가벼운 Todo List를 지향합니다.

------------

![todo_phone_1](https://user-images.githubusercontent.com/67727981/113268268-2ab2fe00-9312-11eb-9117-2b2f8ae60c23.png)

- Firebase를 활용하여 이메일을 통한 회원가입 및 로그인, 구글 로그인 사용

![todo_phone_2](https://user-images.githubusercontent.com/67727981/113268354-40282800-9312-11eb-85c8-e97518ebb0af.png)

- 간단하게 할 일을 추가하고 체크 및 삭제 가능
- 간단한 메모장 역할로도 사용 가능

![todo_phone_3](https://user-images.githubusercontent.com/67727981/113268690-9c8b4780-9312-11eb-8c15-f295b2c41eab.png)

- 입력한 Todo List는 Firebase를 통해 저장되어 날짜별로 기록
- 각 날짜를 클릭하면 그 날에 작성했던 Todo List를 확인 가능
